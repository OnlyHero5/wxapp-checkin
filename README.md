# 微信小程序活动二维码签到

本仓库实现微信小程序前端的活动二维码签到功能，包含注册绑定、扫码验证、签到记录与现代化 UI 方案。

## 当前状态
- 已完成 TDesign（腾讯官方）组件化改造
- 视觉主题为“石墨雾”深色体系，主按钮为深蓝色
- 新增“个人中心”页与 TabBar 入口
- NPM 依赖已迁移至 `src/`（与 `miniprogramRoot` 对齐）

- `design-system/` 视觉与设计系统文档

## 功能概述
- 微信身份与学号姓名绑定
- 滚动二维码（10 秒更新）签到验证
- 签到成功动画与失败提示
- 签到记录列表与详情
- 无网络状态提醒

## 开发与运行
1. 使用微信开发者工具导入仓库根目录。
2. 在 `src/` 目录安装依赖：

```bash
cd src
npm install
```

3. 在微信开发者工具中执行：`工具 → 构建 NPM`。
4. 点击“编译/预览”进行运行验证。
