# 微信小程序活动二维码签到

本仓库用于实现微信小程序前端的活动二维码签到功能，包含注册绑定、扫码验证、签到记录与现代化 UI 方案。

## 文档
- 需求文档：`docs/REQUIREMENTS.md`
- 功能说明书：`docs/FUNCTIONAL_SPEC.md`

## 目录结构
- `docs/` 需求与功能说明
- `README.md` 项目说明

## 功能概述
- 微信身份与学号姓名绑定
- 滚动二维码（10 秒更新）签到验证
- 签到成功动画与失败提示
- 签到记录列表与详情
- 无网络状态提醒
